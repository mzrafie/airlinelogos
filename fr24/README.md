# ✈️ Airline Logos Dataset (FR24)

## 📌 Overview

This repository contains a collection of airline logos gathered from Flightradar24 (FR24) for **educational and non-commercial purposes only**.

The dataset is intended for:

* Learning projects
* UI/UX design practice
* Machine learning / classification experiments
* Aviation-related educational content

---

## 📂 Contents

* Airline logo images (various formats such as PNG/SVG)
* Organized by airline name or ICAO/IATA code (if applicable)
